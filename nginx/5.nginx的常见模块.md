# nginx 常见模块

## 访问控制基于模块ngx_http_access_module实现，可以通过匹配客户端源IP地址进行限制
```bash
location = /login/ {
   root /data/nginx/html/pc;
   allow 10.0.0.0/24;
   deny all;
}

location /about {
   alias /data/nginx/html/pc;
   index index.html;
   deny  192.168.1.1;
   allow 192.168.1.0/24;
   allow 10.1.1.0/16;
   allow 2001:0db8::/32;
   deny all;              #按先小范围在前，大范围在后排序
}

```
